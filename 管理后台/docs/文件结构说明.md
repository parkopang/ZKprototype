# iMDT 管理后台 - 文件结构说明

## 📁 目录结构

```
PRD/管理后台/
├── imdt_admin_index.html              # 主入口文件（骨架结构）
├── iMDT管理平台_原型_v1.0_完整版.html  # 完整版原型（参考用）
├── imdt_admin_index_old.html          # 备份文件
│
├── css/                               # CSS 样式文件目录
│   ├── imdt_admin.css                 # 主样式文件
│   └── imdt_admin.css.old             # 旧版本备份
│
├── js/                                # JavaScript 文件目录
│   ├── app.js                         # 主应用逻辑（页面路由、动态加载等）
│   └── imdt_admin.js                  # 其他JS文件
│
├── pages/                             # 页面HTML片段目录
│   └── home.html                      # 首页内容
│   └── [其他页面...]                  # 其他页面片段
│
└── docs/                              # 文档文件目录
    ├── README.md                      # 项目说明文档
    ├── 登录账号记录.md                # 登录账号信息
    ├── 文件结构说明.md                # 本文件
    └── [其他检查记录和文档...]        # 其他文档
```

## 📋 文件说明

### 主入口文件
- **imdt_admin_index.html** (395行)
  - 主骨架结构
  - 包含顶部导航栏、左侧边栏、主内容区
  - 通过 JavaScript 动态加载页面内容

### CSS 样式
- **css/imdt_admin.css** (1162行)
  - 完整的样式定义
  - 包含所有组件样式（导航、菜单、表格、表单等）

### JavaScript 逻辑
- **js/app.js** (303行)
  - 页面路由管理
  - 动态页面加载
  - Tab 标签管理
  - 菜单交互逻辑

### 页面片段
- **pages/** 目录
  - 每个页面作为独立的 HTML 片段文件
  - 通过 `app.js` 动态加载
  - 每个文件不超过 500 行

### 文档
- **docs/** 目录
  - 所有 Markdown 文档集中管理
  - 包括项目说明、检查记录、账号信息等

## 🔗 文件引用关系

```
imdt_admin_index.html
    ├── 引用: css/imdt_admin.css
    ├── 引用: js/app.js
    └── 动态加载: pages/*.html

js/app.js
    └── 动态加载: pages/*.html
```

## 📝 文件行数限制

- **主HTML文件**：< 1000行 ✅ (当前 395行)
- **JavaScript文件**：< 1000行 ✅ (当前 303行)
- **CSS文件**：不限制（统一管理样式）
- **页面片段**：< 500行 ✅ (当前 15行)

## 🚀 使用说明

1. 直接在浏览器中打开 `imdt_admin_index.html` 即可查看界面
2. 修改样式：编辑 `css/imdt_admin.css`
3. 修改逻辑：编辑 `js/app.js`
4. 添加页面：在 `pages/` 目录下创建新的 HTML 片段文件
5. 查看文档：在 `docs/` 目录下查看相关文档
