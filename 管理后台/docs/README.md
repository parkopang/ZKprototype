# iMDT管理平台原型

## 文件说明

- `iMDT管理平台_原型_v1.0.html` - 后台管理系统原型初稿
- `iMDT管理平台_原型_v1.0_完整版.html` - 后台管理系统原型完整版（推荐使用）

## 原型功能

### 1. 顶部导航栏
- Logo和系统名称
- 字号设置（小号/中号/大号）
- 通知图标
- 全屏按钮
- 用户信息（头像、用户名、下拉菜单）

### 2. 左侧边栏菜单
- 我的首页
- 我的患者
- 讲稿中心
- 会诊中心
- MDT团队
- 共享病例库
- 随访管理
- 收起菜单按钮

### 3. 主内容区
- **Tab标签栏**：支持多标签页切换
- **面包屑导航**：显示当前页面路径
- **内容区域**：
  - **首页**：智能体介绍和快速操作按钮
  - **我的患者**：患者列表（含示例数据、搜索、筛选、创建、编辑、删除、分页）
  - **讲稿中心**：讲稿卡片列表（含示例数据、搜索、筛选、创建、查看、编辑、分页）
  - **会诊中心**：会诊列表（含示例数据、搜索、筛选、创建、查看、编辑、开始、加入、分页）
  - **MDT团队**：团队卡片列表（含子标签：我的MDT团队/全部MDT团队、团队统计信息、搜索、筛选、创建、查看、管理、加入、分页）
  - **共享病例库**：病例表格列表（含示例数据、搜索、筛选、共享、病例详情、会诊详情、分页）
  - **随访管理**：随访记录列表（含示例数据、搜索、筛选、创建、查看、编辑、分页）

## 使用说明

### 完整版（推荐）
1. 直接在浏览器中打开 `iMDT管理平台_原型_v1.0_完整版.html` 文件
2. 点击左侧菜单可以切换不同页面
3. 每个页面会以Tab标签的形式打开，支持多标签页
4. 可以点击Tab标签的关闭按钮关闭页面
5. 所有页面都包含详细的示例数据
6. 表格和卡片列表都支持分页功能
7. 点击"创建"按钮可以打开创建表单（模态框）
8. 所有操作按钮都有对应的交互提示

### 初稿版
1. 直接在浏览器中打开 `iMDT管理平台_原型_v1.0.html` 文件
2. 基础功能演示，包含首页、我的患者、会诊中心三个页面

## 设计特点

- **现代化UI**：采用蓝白色调，清爽简洁
- **响应式布局**：固定顶部导航和侧边栏，主内容区自适应
- **多标签页**：支持同时打开多个页面，方便切换
- **交互友好**：菜单项、按钮都有hover效果
