## 📝 补全步骤（每个模块）

1. **提取页面内容**
   - 从完整版HTML中找到对应的 `id="page-xxx"` 部分
   - 提取完整的HTML结构（去除外层div和id）

2. **创建页面文件**
   - 创建 `pages/xxx.html` 文件
   - 确保文件行数 < 500行

3. **提取JavaScript函数**
   - 查找相关的 `function xxx()` 函数
   - 添加到 `js/app.js` 中

4. **更新页面映射**
   - 确保 `js/app.js` 中的 `pageTitles` 包含该页面
   - 确保菜单中的 `data-page` 正确

5. **运行构建脚本**
   - 运行 `node build-pages.js` 更新 `pages-loader.js`

6. **验证**
   - 打开浏览器测试页面加载和功能
